# Git & GitHub Resources

> Bilim shu yerdagi ishonchli manbalardan olinadi (xotiraga emas). Donolik (wisdom)
> esa quyidagi jamoalardan — real savol-javob va amaliyotdan keladi.

## Knowledge (Bilim)

### Kanonik / inglizcha
- [Kitob: **Pro Git** — Scott Chacon & Ben Straub (git-scm.com)](https://git-scm.com/book/en/v2)
  Git'ning rasmiy, bepul va eng ishonchli kitobi. **Eng asosiy manba.** Use for:
  har qanday tushuncha, buyruq, ichki ishlash tartibi. (CC BY-NC-SA litsenziya.)
- [Git rasmiy hujjatlari / Reference](https://git-scm.com/doc)
  Har bir buyruqning to'liq ta'rifi (`git help <buyruq>` ham shu). Use for: aniq flag/option qidirish.
- [**GitHub Skills** (interaktiv kurslar)](https://skills.github.com/)
  GitHub'ning o'zida, real Issue/Action ichida bajariladigan amaliy mashqlar.
  Use for: "Introduction to GitHub", birinchi pull request, real GitHub feel.
- [GitHub Docs — Git & GitHub learning resources](https://docs.github.com/en/get-started/start-your-journey/git-and-github-learning-resources)
  Use for: GitHub'ga oid aniq amallar (akkaunt, SSH, commit email sozlash).
- [**Learn Git Branching** (vizual, interaktiv o'yin)](https://learngitbranching.js.org/)
  Branch/merge/rebase'ni jonli commit-daraxti bilan ko'rsatadi. Use for: **branching**ni
  ko'z bilan ko'rib tushunish (bizning maqsad uchun juda muhim).

### O'zbekcha
- [**Git Ustasi** — gitustasi.uz](https://gitustasi.uz)
  Bepul, o'zbekcha, **brauzerda terminal** bilan interaktiv 15+ dars. Yo'nalishi
  aynan bizniki: asoslar → buyruqlar → branching/merging → jamoada ishlash → professional.
  Use for: har bir darsdan keyin o'zbekcha amaliy mashq. **Asosiy o'zbekcha hamroh.**
- [xteam-uz/github-darslari (GitHub'da o'zbekcha qo'llanma)](https://github.com/xteam-uz/github-darslari)
  Use for: GitHub bo'yicha o'zbekcha matnli qo'llanma.
- [O'zbek tilida dasturlash resurslari to'plami](https://github.com/elzodxon/programmingResourcesUzbek)
  Use for: umumiy o'zbekcha o'quv manbalar (Git'dan tashqari ham).

## Wisdom (Communities — jamoalar)

- [**Stack Overflow** — `git` tegi](https://stackoverflow.com/questions/tagged/git)
  Yuqori moderatsiyali, dunyodagi eng katta Q&A. Use for: aniq xato/savolga javob izlash
  (avval qidiring — ehtimol javob bor).
- [r/git (Reddit)](https://www.reddit.com/r/git/)
  Git'ga ixtisoslashgan, sifatli subreddit. Use for: workflow maslahati, muammoni tushuntirish.
- [O'zbek dev Telegram guruhlari ro'yxati (awesome-telegram-dev-groups-uz)](https://github.com/doniyor2109/awesome-telegram-dev-groups-uz)
  Use for: o'zbekcha so'rab-surishtirish uchun faol guruh tanlash.
- [Uzbek Coders hamjamiyati](https://community.uzbekcoders.uz/)
  Use for: boshlovchi o'zbek dasturchilar bilan muloqot, mentoring.

## Gaps (Yetishmayotgan narsalar)
- Pro Git kitobining **to'liq o'zbekcha** tarjimasi topilmadi (turkcha bor, o'zbekcha yo'q).
  Shu sababli inglizcha Pro Git + o'zbekcha gitustasi.uz juftligidan foydalanamiz.
- Foydalanuvchi jamoaga qo'shilishni xohlamasligini bildirmadi — kerak bo'lsa yuqoridagilardan birini tavsiya qilamiz.
